Salida: alert() y console

Algo muy normal al momento de hacer, por ejemplo debug de tu codigo, es usar a un personaje muy famoso en el mundo de js, como es console ya que con este objeto, podemos imprimir en la consola del navegador cualquier tipo de dato, esto lo hacemos usando la palabra reservada console seguido de su propiedad (hay muchas, en los ejemplos mostramos las tres mas comunes) y a continuacion el o los datos.

Si console nos ayuda, por ejemplo a hacer debug, alert() nos ayuda a comunicar mensajes que no requieren ninguna respuesta por parte del usuario, excepto el reconocimiento del mensaje. 

/* Alert */
const mensaje = 'Hola mundo!!!';
alert(mensaje);
alert('Otro hola mundo!!!');
/* Console */
const logVar = 'Log'; 
console.log(logVar);
console.error('error', 'otro error', logVar);
console.warn('warning!!!');
                
/* Prompt */
const nombre = prompt('Cual es tu nombre?');
const curso = parseInt(prompt('Y tu curso?'));
console.log('Oh! un gusto ' + nombre + '!!!');
console.log('Bienvenido a ' + curso + '!!!');
/* Confirm */
const eliminar = confirm('Desea eliminar el registro?');
if (eliminar) {
  console.log('Eliminado');
} else {
  console.log('No eliminado');
}

Entrada: prompt() y confirm()

Para la entrada de datos del lado del usuario, para empezar, podemos usar la funcion prompt() la misma, a diferencia del alert, añade un text field para ingresar datos de manera opcional, luego los retorna para poder manipular ese dato dentro de nuestro script

Nota: es importante saber que prompt retorna siempre un string y si se desea manipular numeros con prompt va a ser necesario castear el dato retornado. En el caso del ejemplo se muestra el casteo con parseInt() a un tipo de dato entero.

A menudo en determinados casos es necesario que el usuario verifique o acepte algo. Para estos casos podemos utilizar el metodo confirm() que muestra un cuadro de diálogo con un mensaje específico, junto con un botón "Aceptar" y un botón "Cancelar". Este metodo recibe el mensaje y retorna true si el usuario hizo clic en "Aceptar" y false en caso contrario.
